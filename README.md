# BirthdayFunFacts

## BirthdayFunFacts is an entertaining website that not only inspires you but gives you interesting information about your birthday. 

* The application is for entertainment purposes and can be shared with friends and family members for fun on their birthday
* The application utilizes 2 Server-Side APIs
  *  One API provides a listing of motivational quotes
  *  One API provides a listing of historical events that occured on each day of the year
*  The application utilizes the CSS framework, Bulma
